It manages trainees and materials (veneer, solid wood, engineered panels, fittings), auto-calculates areas, waste allowances and costs, and generates the full project paperwork — material lists, cut list, cost summary, workflow plan and evaluation sheet, each with barcodes. React + zustand + styled-components on a **Tauri** shell with a **Rust + SQLite** backend; fully keyboard-driven; requirements gathered on-site **in German** directly from the workshop master.
